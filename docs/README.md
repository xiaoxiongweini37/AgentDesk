# AgentDesk 项目文档

本目录包含 AgentDesk 项目的设计文档、实现说明和调研资料。

## 目录结构

```
docs/
├── README.md                    # 本文档
├── design.md                    # 项目设计文档
├── implementation.md            # 实现细节
├── research.md                  # 调研资料
├── api.md                       # API 接口文档
├── architecture.md              # 架构设计
└── development.md               # 开发指南
```

## 快速导航

- **新成员入门**: 阅读 `design.md` 了解项目背景和目标
- **开发者**: 阅读 `implementation.md` 和 `development.md`
- **API 集成**: 阅读 `api.md`
- **架构理解**: 阅读 `architecture.md`

## 文档维护

所有文档使用 Markdown 格式编写，遵循以下规范：
- 使用中文撰写
- 保持简洁明了
- 代码示例使用正确的语法高亮
- 图表使用 Mermaid 或 ASCII 艺术
